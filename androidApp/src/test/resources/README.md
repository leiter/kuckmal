# Test Resources

## Filmliste-akt

This directory should contain the `Filmliste-akt` file for testing purposes.

**Note:** This file is too large to commit to git and is excluded via `.gitignore`.

### How to obtain the file:

The `Filmliste-akt` file can be obtained from:
- MediathekView servers: https://verteiler1.mediathekview.de/Filmliste-akt.xz
- Extract the .xz file after downloading

### Usage:

Tests that require this file will reference it from this directory.
Make sure the file exists here before running tests that depend on it.
