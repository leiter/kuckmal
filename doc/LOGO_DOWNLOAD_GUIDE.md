## Download Instructions

### From Wikimedia Commons:
1. Click the Wikimedia link for your desired logo
2. Look for the file preview on the page
3. Click "More details" or "Use this file"
4. Choose "Download" and select PNG format
5. Choose size: **300px or 500px width** recommended
6. Save with the exact filename listed above

### From SeekLogo/VectorSeek:
1. Click the link
2. Click "Download" button
3. Choose PNG format (or SVG if you want to convert yourself)
4. Select "High Quality" or "Original" size
5. Save with the exact filename listed above

### From IconApe:
1. Click the link
2. Scroll to download section
3. Choose PNG format
4. Select size (300-500px recommended)
5. Click download and save with correct filename

## Post-Download Processing

After downloading, you may want to optimize the logos:

### Option 1: Use the resize script
I can create a Python script to batch resize and optimize all downloaded logos.

### Option 2: Manual processing
- **Resize:** Keep aspect ratio, target width 200-400px
- **Format:** Ensure PNG with transparency
- **Optimize:** Use tools like pngcrush or optipng to reduce file size
- **Location:** Place in `app/src/main/res/drawable/`

If you'd prefer, you can also:
1. Download logos manually to a folder (e.g., `tmp/logos/`)
2. I can create a script to batch process them (resize, optimize, rename)
3. Move them to the correct drawable folder
